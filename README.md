# Overview

## This project is a proof-of-concept internal AI-powered query assistant developed using Python, Flask, and SQLite.

## The system simulates an internal telecommunications support tool capable of retrieving:

Open and closed ticket status

Wireless/mobile service issues

Internal department contact information

Service-related query responses

## Technologies Used

Python

Flask

SQLite

HTML

VS Code

## Features

Natural-language style ticket queries

Internal directory lookup

Web-based interface

SQL-backed ticket database

Simple AI-style query processing

## Example Queries
Show open tickets,

Find wireless issues,

Show closed tickets,

Who is the IT contact?
